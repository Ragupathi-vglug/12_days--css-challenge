# 🎨 12 Days CSS Challenge

Welcome to my **12 Days CSS Challenge** 🚀  

This repository contains my journey of learning and mastering **CSS (Cascading Style Sheets)** by building projects and exploring concepts over 12 days.

---

## 🎯 Goal

- Master CSS from basics to advanced  
- Build strong UI/UX design skills  
- Practice daily with real examples  
- Create responsive and modern web designs  

---

## 📅 Challenge Overview

Each day focuses on specific CSS concepts and practical implementations.

---

## 🧠 Topics Covered

### 🟢 Day 1 – CSS Basics
- Introduction to CSS  
- Inline, Internal, External CSS  
- Selectors  
- Colors & Units  

---

### 🟢 Day 2 – Box Model
- Margin, Padding  
- Border  
- Width & Height  
- Box-sizing  

---

### 🟢 Day 3 – Typography
- Fonts & Font Families  
- Font Size, Weight  
- Text Alignment  
- Line Height & Spacing  

---

### 🟢 Day 4 – Colors & Backgrounds
- Background Colors  
- Background Images  
- Gradients  
- Opacity  

---

### 🟡 Day 5 – Flexbox
- Flex Container  
- Flex Items  
- Justify Content  
- Align Items  
- Flex Direction  

---

### 🟡 Day 6 – CSS Grid
- Grid Container  
- Rows & Columns  
- Grid Areas  
- Gap  

---

### 🟡 Day 7 – Positioning
- Static, Relative  
- Absolute, Fixed  
- Z-index  

---

### 🟡 Day 8 – Responsive Design
- Media Queries  
- Mobile-first Design  
- Flexible Layouts  

---

### 🔴 Day 9 – Animations
- Transitions  
- Keyframes  
- Hover Effects  

---

### 🔴 Day 10 – Advanced CSS
- Pseudo-classes  
- Pseudo-elements  
- Variables (`:root`)  

---

### 🔴 Day 11 – Modern UI Design
- Cards Design  
- Glassmorphism  
- Shadows & Effects  

---

### 🔴 Day 12 – Final Projects
- Landing Page  
- Portfolio UI  
- Complete Responsive Design
